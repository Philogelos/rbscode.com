Expanded Examples (Read This Slowly)

0. Colors (rgb, rgba, hex)

CSS can understand colors in a few different ways. All of these are valid, and you can choose whichever feels easiest.

/* 1) Hex (short color code) */
color: #1F214D;

/* 2) rgb(red, green, blue) - numbers from 0 to 255 */
background-color: rgb(30, 30, 60);

/* 3) rgba(red, green, blue, alpha) - alpha is transparency */
box-shadow: 0 8px 18px rgba(0, 0, 0, 0.18);

rgb() uses 3 numbers (0–255):
rgb(255, 0, 0) is bright red, and rgb(0, 0, 0) is black.

rgba() is the same as rgb, but with a 4th number called alpha (see-through):
rgba(0,0,0,1) = fully solid
rgba(0,0,0,0.2) = very transparent

Hex is a short code that starts with #.
It’s just another way to write colors, like a color nickname.

1. Gradients (Backgrounds with Multiple Colors)

A gradient blends colors smoothly. The simplest version looks like this:

background: linear-gradient(to bottom, #1F214D, #BF3475, #FFCE61);

Common directions you can use:

  • to bottom
  • to top
  • to right
  • to left
  • to top right
  • to bottom right

You can use 2 colors or 3 colors. More than that is usually unnecessary.

2. Padding and Margin (Spacing)

Padding = space inside the box (content moves inward)
Margin = space outside the box (the box moves away from other things)

/* ONE number = all sides */
padding: 10px;

/* TWO numbers = vertical, horizontal */
/* first = top+bottom, second = left+right */
padding: 10px 20px;

/* FOUR numbers = top, right, bottom, left */
padding: 10px 0 30px 20px;

/* You can also control one side */
padding-left: 5px;

margin: 20px;
margin: 10px 30px;
margin: 10px 0 30px 20px;

Remember the 4-number order like a clock: top → right → bottom → left (12 o’clock → 3 o’clock → 6 o’clock → 9 o’clock)

3. Box Shadow (Depth)

A shadow makes a card feel lifted off the page.

box-shadow: 0 8px 18px rgba(0,0,0,0.18);

What each part means:

  • 0 = move the shadow left or right (negative goes left, positive goes right)
  • 8px = move the shadow up or down (negative goes up, positive goes down)
  • 18px = blur amount (bigger number = softer shadow)
  • color = shadow color (often black with transparency)

Tip: Most of the time you want a small x, a medium y, and a soft blur.

4. Aspect Ratio (Card Shape)

Aspect ratio locks the shape of a card. This is useful when you want all cards to be the same shape.

aspect-ratio: 3 / 4;

This means: for every 3 units wide, the card is 4 units tall.

Extra important rule: you only need to set width OR height (not both). The browser will choose the other size automatically to keep the ratio.
