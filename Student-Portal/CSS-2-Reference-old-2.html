Special Topic — display

The display property decides how a box behaves: does it sit on its own line, act like text, or behave like a card in a row?

ValueWhat it means
blockA box on its own line (default for div)
inlineActs like text (no width / height)
inline-blockA box that sits in a line (perfect for cards)
Example: Cards in a row

.card { display: inline-block; }
Card 1
Card 2
Card 3

Key idea:
Inline elements act like text.
Block elements act like big boxes.
Inline-block elements are boxes that can sit next to each other.
