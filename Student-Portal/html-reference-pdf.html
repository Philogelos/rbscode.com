HTML Reference — Lesson 1

This reference sheet shows the essential HTML tags taught in Lesson 1. A short quick-reference appears at the end.

Heading Tags — <h1> to <h6>

Used for titles and subtitles. <h1> is the largest heading and <h6> is the smallest.

<h1>Main Title</h1>
<h3>Subtitle</h3>

Main Title

Subtitle

<p> — Paragraph

Used for blocks of text.

<p>This is a paragraph of text.</p>

This is a paragraph of text.

<a> — Link

Creates a clickable link to another page.

<a href="https://example.com">Visit Page</a>
Visit Page

<img> — Image

Displays an image. Uses src, alt, and width.

<img src="hero.png" width="150" alt="Hero">

<button> — Button

A clickable button.

<button>Click Me</button>

Lists — <ul>, <ol>, <li>

<ul>: bullet list
<ol>: numbered list
<li>: list item 

<ul>
  <li>Speed: 90</li>
  <li>Stealth: 97</li>
</ul>
  • Speed: 90
  • Stealth: 97

<br> — Line Break

Moves the next text to a new line.

Line 1<br>Line 2
Line 1
Line 2

<hr> — Horizontal Line

Creates a dividing line.

<hr>
